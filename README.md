# My Favorite Links

A simple React + Vite application to manage your favorite links.

## Features

- Add new links with a name and URL
- View all links in a table format
- Remove links you no longer need
- Links are persisted in localStorage

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Usage

1. Enter a link name and URL in the form
2. Click "Submit" to add the link to the table
3. Click "Remove" next to any link to delete it
4. Click on any URL to open it in a new tab
